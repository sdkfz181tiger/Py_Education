
1, SSH有効
	SDカードドライブ直下に"ssh"ファイルを作る

	touch /Volumes/boot/ssh

2-1, 接続先Wifi
	SDカード直下に"wpa_supplicant.conf"を作る

	touch /Volumes/boot/wpa_supplicant.conf

2-2, Wifiの接続先を記述

	vim /Volumes/boot/wpa_supplicant.conf

	wpa_supplicant.conf記述内容

	ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
	update_config=1
	country=JP

	network={
	    ssid="**********"
	    psk="**********"
	    key_mgmt=WPA-PSK
	    priority=2
	}

3-1, SSHアクセス

	ssh pi@raspberrypi.local

3-2, 再度アクセスする場合、該当のhostを削除してからアクセスする

	sudo vim ~/.ssh/known_hosts

4-1, localeの設定

	-> localeの設定
		nano ~/.bashrc

	-> ファイルの最下に下記を追記(ctrl + xで終了 -> Yes)
	export LANGUAGE=en_GB.UTF-8
	export LC_ALL=en_GB.UTF-8
	export LC_TYPE=en_GB.UTF-8

	-> 適用してリブート
		sudo locale-gen
		sudo reboot now

	-> 確認
		locale

4-2, パッケージのUpdate

	sudo apt-get update
	sudo apt-get upgrade
	sudo apt-get dist-upgrade

4-3, ファームウェア更新

	sudo rpi-update

4-3, Python3

	sudo apt-get -y install python3-dev
	sudo apt-get -y install python3-pip

4-4, Chrome

	sudo apt-get install chromium-browser
	sudo apt-get install chromium-chromedriver

	調子が悪い場合は一旦アンインストール
		sudo apt-get remove chromium-browser
		sudo apt-get remove chromium
		sudo apt-get autoremove
		sudo apt-get clean

	バージョン確認
		chromium-browser --version
		chromedriver --version

	sudo pip3 install selenium

5, Shell

	hoge.shファイルを作り、コマンドを記述
		#!/bin/bash
		cd /home/pi/Desktop/
		/usr/bin/python3 main.py

	シェルを有効にする
		chmod +x hoge.sh

6, Cron

	Cron編集
	Crontab -e

	# KoboCheck -> HatenaBlog
	00 19 * * * /home/pi/Desktop/hoge.sh

	Cron確認
	Crontab -l

	Cronログ確認可能にするには下記ファイル#cronを#削除
	sudo vim /etc/rsyslog.conf

	sudo cat /var/log/cron.log
